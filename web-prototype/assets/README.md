# Web Prototype Assets

这里用于放网页版原型素材。当前仓库先记录素材用途，后续实现时再把图片文件放进对应目录。

建议目录：

```text
reference/      参考图，不直接在最终页面中裁切使用
backgrounds/    公园地图、街景、区域背景
characters/     主鸟、其他鸟类、羽毛
ui/             软木板、便签、图标、按钮、线索线
```

已确认可用素材方向：

- 五栏鸟类记忆视觉：用于记忆类型和视觉语言。
- 主鸟三视图：用于标题页和 UI 角色头像。
- 俯视公园地图：用于主导航地图。
- 公园街景：用于首页和区域页面背景。
